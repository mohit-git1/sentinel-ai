## Design Context

### Users
Primary users are solo developers reviewing pull requests in GitHub workflows. They use Sentinel to reduce missed issues before merge while keeping momentum high.

### Brand Personality
Confidence-first, clear, and premium. The interface should feel trustworthy and composed with strong information hierarchy, minimal noise, and purposeful interactions.

### Aesthetic Direction
Minimal, clean, modern product UI with neutral tones and reduced visual effects. Avoid AI-generic motifs: purple gradients, glassmorphism, heavy glow/shadows, and over-animated surfaces. Default to dark theme with a first-class light mode.

### Design Principles
1. Prioritize clarity over decoration: each visual element must serve comprehension.
2. Use restrained neutral color scales with accent only for state and action hierarchy.
3. Build rhythm with spacing and typography contrast instead of large effects.
4. Keep interaction feedback subtle, fast, and accessible (including reduced motion paths).
5. Maintain visual consistency across authentication, dashboard, and detail workflows.
